# Mini-Brief Web : Landing Page Love4You

## 1. Objectif du site
- **Rôle principal :** Convertir les visiteurs en utilisateurs de l'application mobile.
- **Action attendue (Call-To-Action) :** Cliquer sur les boutons "Télécharger sur l'App Store" ou "Disponible sur Google Play".
- **Rôle secondaire :** Héberger les pages légales (Politique de confidentialité et CGU), ce qui est obligatoire pour publier une application sur les stores.

## 2. Design & Ton
- **Ambiance :** Sombre (Dark mode natif), élégant, romantique et mystérieux (ex: fond `neutral-950` avec des accents de couleur néon/rose/rouge selon le branding de l'app).
- **Ton :** Complice, ludique, sans jugement.

## 3. Structure du site (Sitemap)
Le site sera composé de 3 pages maximum pour la V1 :
1. **`/` (Page d'accueil) :** - Hero Header (Titre accrocheur + mockup téléphone + 2 boutons de téléchargement).
   - Section "Pourquoi Love4You ?" (Les problèmes résolus).
   - Section "Fonctionnalités" (Fantasies, Daily Action, Love Quiz, Sexy Dice en grille).
   - Footer (Liens légaux, contact).
2. **`/privacy` :** Page de texte simple pour la politique de confidentialité.
3. **`/terms` :** Page de texte simple pour les conditions d'utilisation.

## 4. Stack Technique
- **Framework :** Next.js (App Router, TypeScript).
- **Styling :** Tailwind CSS.
- **Animations :** Framer Motion (pour de légères apparitions au scroll).
- **Icônes :** Lucide React.
- **Hébergement prévu :** Vercel (déploiement continu depuis GitHub).
